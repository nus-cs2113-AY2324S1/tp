Functions and their format:

Adding an entry: add
Format:
add income /de <description> /date <date> /amt [currency] <amount>
add expense /cat <category> /type <type> /de <description> /date <date> /amt <amount>

Listing all entries: list
Format:
list
list income
list expense
list currencies
list exchange rates

Deleting an entry: delete
Format:
delete income <index_pos>
delete expense <index_pos>

Edit an entry: edit
Format:
edit income <index> /de <description> /date <date> /amt [currency] <amount>
edit expense <index> /cat <catergory> /type <type> /de <description> /date <date> /amt [currency] <amount>

Check balance of income: balance
Format: balance

Update exchange rate: update exchange rate
Format: update exchange rate <supported_currency> <rate>

Finding an entry: find
Format:
find /t income /de [description] /date [date]
find /t expense /cat [category] /type [type] /de [description] /date [date]

Clearing all entries: clear
Format:
clear income/expense
clear all

Exiting the program: exit
Format: exit
